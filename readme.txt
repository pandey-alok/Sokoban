************************How To Run***************************************************
1. Keep all the test cases file in the folder: "testCases"
2. Rename all the test case files with names - testExample1.xsb, testExample2.xsb, testExample3.xsb and so on.
3. Use command "python3 sokoban.py <number of the testcases in the folder>"
4. Solutons of the test cases will be created in the same "testCase" folder and will be named as Solution_1.xsb, Solution_2.xsb, Solution_3.xsb and so on.
